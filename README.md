# How to run tailwindcss
1. truy cập link: https://github.com/tailwindlabs/tailwindcss/releases
2. download bản tailwindcss-windows-x64.exe
3. đổi tên thành tailwindcss.exe
4. add vào folder Group-3_Warehouse-Management-System
5. vào terminal cd  Group-3_Warehouse-Management-System
6. chạy lệnh: ./tailwindcss.exe -i ./src/main/webapp/css/input.css -o ./src/main/webapp/css/style.css --watch
